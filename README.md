# Возможности
-  **Список рецептов** — просмотр всех рецептов
-  **Поиск** — фильтрация по названию 
-  **Фильтры** — по статусу: Все / Хочу приготовить / Готовлю / Приготовлено
-  **Статистика** — количество рецептов в каждом статусе
-  **Экран деталей** — полная информация о рецепте
-  **Изменение статуса** — переключение по кругу с любого экрана
-  **Навигация** — переход между экранами с сохранением состояния
